Visual Rules do CSS

Propriedades do texto Descrição
Tipo de Letra. Define a Font a ser usada no texto do elemento a que se refere.
Tamanho de Letra Determina o tamanho do texto apresentado
Espessura da letra Determina a espessura do texto apresentado (Bold ou Normal)
Opacidade Define Quão Transparente é o elemento.

Propriedades CSS

Propriedade CSS Modo de utilização
font-family Permite selecionar o tipo de letra do texto da página. O tipo de letra selecionado deve estar instalado, ou devem ser usadas Web Safe Fonts.
font-size Permite selecionar o tamanho de letra, normalmente é medido em Pixels.
font-weight Permite selecionar a espessura de letra, pode ter 2 valores: Bold ou Normal.
opacity Permite selecionar a Opacidade de um elemento, e tem valor entre 0 e 1, em que 0 é completamente invisível e 1 é completamente opaco.
